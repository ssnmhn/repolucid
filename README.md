## Why this exists

In 2026, 30+ coding tools read `AGENTS.md`. Readiness scanners will tell you yours is missing —
then generate boilerplate. But a context file that doesn't encode your actual architecture,
invariants, and sharp edges makes agents *confidently wrong* instead of obviously blind. The
same gap is what makes every new engineer's first month expensive.

Scanners check that files exist. We write what goes in them.
